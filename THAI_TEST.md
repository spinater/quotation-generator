## Troubleshooting

### If Thai text shows as boxes (‚ñ°‚ñ°‚ñ°)

**Problem:** Font not loaded
**Solution:**
1. Check console for errors (F12)
2. Verify fonts in `public/fonts/` directory
3. Clear cache: Ctrl+Shift+R
4. Rebuild: `npm run build`

### If vowels are in wrong position

**Problem:** Font doesn't support complex Thai rendering
**Solution:**
- Already fixed with Noto Sans Thai
- This font properly handles Thai character composition

### If tone marks overlap text

**Problem:** Font metrics issue
**Solution:**
- Switch to THSarabunNew font
- Edit `src/utils/fonts.ts`
- Change: `export const FONT_FAMILY = "THSarabunNew";`
